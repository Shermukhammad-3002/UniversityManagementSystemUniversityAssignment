# University Management System

A simple university management system built as a C++ assignment at TMC.

## About
During Year 1 and Year 2 at TMC, I learned several programming languages. This is a console app I built in C++ that simulates the core functionalities of a university management system in a straightforward way.

## Technologies Used
C++

## How to Run
No frameworks or external libraries needed — just clone the repository, open it in any C++ IDE (e.g. Visual Studio, CLion, or Code::Blocks), and run the main file.
